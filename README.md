Projeto desenvolvido para fins de treinamento em ReactJs.

## Pacotes extras utilizadas

- history v5.0.0
- react-router-dom v6.0.0-beta.0

### `Caracteristicas`

- Foi utilizado Hooks:
  - React.useEffect()
  - React.useState()

- Gerenciador de pacotes
  - Yarn
 
- Estilo
  - Nativo do React - arquivo.module.css
